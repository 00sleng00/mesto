# Проект: Mesto

## О проекте:
Mesto -это интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.


#### Технологии
+ HTML5
+ CCS3
+ JavaScript
+ Адаптивная верстка
+ Методология БЭМ (файловая структура)
+ Flexbox
+ GRID
+ Позиционирование
+ Псевдоклассы
